Write stubs for these requirement IDs (use the exact ID as part of the test title string so `-g "NAV-01"` grep works):

```
NAV-01 — page.locator('nav[aria-label="Site sections"]') has 6 anchor children; check hrefs #hero, #about, #experience, #projects, #skills, #contact
NAV-02 — nav contains diamond character (◆ or ◇) in text content
HERO-01 — section#hero visible; contains text matching /LVL \d+/ (level bar label)
HERO-02 — section#hero contains a portrait placeholder element (class or data attr indicating portrait)
HERO-03 — section#hero contains a heading and paragraph text outside .trainer-card (or equivalent wrapper)
HERO-04 — section#hero contains two links with href="#projects" and href="#contact"
ABOUT-01 — section#about is visible
ABOUT-02 — section#about contains .panel element
EXP-01 — section#experience contains an ordered or unordered list with at least 1 li
EXP-02 — first experience entry contains company name text, role text, date text, and at least 1 achievement li
EXP-03 — section#experience contains an element with class is-current
PROJ-01 — section#projects contains .projects-grid with at least 2 .panel children
PROJ-02 — first project card contains text matching /#\d{3}/ (dex number), a project name heading, and two links
PROJ-04 — project cards have class .panel
SKILL-01 — section#skills contains .skills-grid with at least 4 .skill-slot children
SKILL-02 — section#skills contains element with role="tablist"
SKILL-03 — first .skill-slot contains an svg or img element (icon) and a visible text label
CONT-01 — section#contact heading contains text "Rest Area / Save Point" (exact)
CONT-02 — section#contact contains links with href matching mailto:, github.com, linkedin.com
A11Y-01 — page has <main>, <nav>, and <section> elements; page.locator('main') count >= 1
A11Y-03 — all Icon elements inside sections have aria-hidden="true" (query svg[aria-hidden="true"] count > 0)
```

Also add `scroll-behavior: smooth` to the `html` selector in `src/styles/global.css`, with a `prefers-reduced-motion` override:
```css
html {
  scroll-behavior: smooth;
}

  html {
    scroll-behavior: auto;
  }
}
```

Run the new tests to confirm they ALL fail (red). A passing test here would indicate incorrect assertions.
