Template — semantic tab markup (ARIA-correct even though JS filtering deferred to Phase 3):
```astro
<section id="skills" aria-labelledby="skills-heading">
  <h2 id="skills-heading" class="section-heading">Inventory</h2>

  <!-- Tab list — JS filter wired in Phase 3; all skills visible by default -->
  <div role="tablist" aria-label="Skill categories" class="tab-list">
    {tabs.map((tab, i) => (
      <button
        role="tab"
        aria-selected={i === 0 ? 'true' : 'false'}
        aria-controls="skills-panel"
        id={`tab-${tab.id}`}
        class:list={['tab-btn', { 'tab-btn--active': i === 0 }]}
        data-category={tab.id}
      >
        {tab.label}
      </button>
    ))}
  </div>

  <!-- Tab panel -->
  <div role="tabpanel" id="skills-panel" aria-labelledby="tab-all">
    <ul class="skills-grid" aria-label="Skills inventory">
      {skills.map((skill) => (
        <li
          class="skill-slot"
          data-category={skill.category}
          data-tooltip={skill.tooltip}
          tabindex="0"
          aria-label={`${skill.name} — ${skill.tooltip}`}
        >
          <Icon name={skill.icon} width="32" height="32" aria-hidden="true" />
          <span class="skill-label">{skill.name}</span>
        </li>
      ))}
    </ul>
  </div>
</section>
```
