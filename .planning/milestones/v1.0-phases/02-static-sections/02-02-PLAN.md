Render as `<nav class="journey-nav" aria-label="Site sections">` with a `<ul>` of `<li>` items. Each `<li>` contains an `<a href="#[id]">` with:

- `aria-current="location"` on the active item
- `class:list={['nav-marker', { 'is-active': s.defaultActive }]}`

```js
const hero = {
  name: '[Your Name]',

```

Structure:
```
<section id="hero" aria-labelledby="hero-heading">
  <div class="hero-layout">
    <!-- Left: headline + tagline (outside the trainer card) -->

      <p class="hero-tagline">...</p>
      <div class="hero-cta">
        <a href="#projects" class="cta-btn">View Projects</a>
        <a href="#contact" class="cta-btn cta-btn--secondary">Contact</a>
      </div>
    </div>
    <!-- Right: trainer card (Panel-wrapped) -->
    <Panel class="trainer-card">
      <!-- Portrait placeholder: square with initials or styled box -->

        <span class="portrait-initials">?</span>
      </div>
      <!-- Stats column -->
      <div class="trainer-stats">
        <div class="stat"><span class="stat-label">NAME</span><span class="stat-value">{hero.name}</span></div>
        <div class="stat"><span class="stat-label">CLASS</span><span class="stat-value">{hero.class}</span></div>
        <div class="stat level-stat">
          <span class="stat-label">LEVEL</span>
          <div class="pixel-bar" role="progressbar" aria-valuenow={hero.level} aria-valuemin={0} aria-valuemax={hero.maxLevel} aria-label={`Level ${hero.level} of ${hero.maxLevel}`}>
            <div class="pixel-fill" style={`width: ${levelPct}%`}></div>
          </div>
          <span class="level-number">LVL {hero.level}</span>
        </div>
        <div class="stat"><span class="stat-label">REGION</span><span class="stat-value">{hero.region}</span></div>
      </div>
    </Panel>
  </div>
</section>
```

Key CSS rules:
- `.hero-layout`: flex row on desktop (intro left, card right), flex col on mobile
- `.portrait-placeholder`: 120×120px square, border: 4px solid var(--color-border), display flex, align-items center, justify-content center, background: var(--color-world)
- `.portrait-initials`: DotGothic16 font, color: var(--color-surface), font-size: 2rem
- `.pixel-bar`: width 120px, height 12px, border: 2px solid var(--color-border), background: var(--color-world)
- `.pixel-fill`: height 100%, background: var(--color-accent-primary)
- `.stat-label`: DotGothic16 font, font-size 0.65rem, color: var(--color-accent-secondary), letter-spacing 0.1em
- `.cta-btn`: Panel-style border (2px solid var(--color-border)), padding 0.5rem 1.5rem, DotGothic16 font, background transparent, color var(--color-text); `:focus-visible` outline: 3px solid var(--color-accent-primary)
- Section heading (outside panel, on world bg): use `color: var(--color-surface)` NOT `--color-text`
